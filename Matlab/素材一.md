## 关系运算

6种关系运算符：<、<=、>、>=、==、~=。

对应的6个函数：lt()、le()、gt()、ge()、eq()、ne()。

## 逻辑运算

3种逻辑运算：&(与)、|(或)、~(非)。
4个逻辑运算函数：and(a,b)、or(a,b)、not(a)和xor(a,b)。
这两种都可以用，是一样的；

```
y=and(x>0,x<10)
y=x>0&x<10 
```

## 相关函数

因式分解与展开
①factor(s)：对符号表达式s分解因式。
②expand(s)：对符号表达式s进行展开。
③collect(s)：对符号表达式s合并同类项。
④collect(s,v)：对符号表达式s按变量v合并同类项。

## 计算

计算一个符号变量组成的函数的值。
subs(f,x,y) 把f中的x换成y
subs(f,y) 把f中的主变量换成y。
symvar(f,1) 找出f的主变量。
eval(f) 重新计算f的值。

```
syms x
m=2^x
x=10
m19=eval(m)
```
